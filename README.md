#  voitheia
A text-based virtual assistant for macOS.
## Syntax
### Send a message
Tell `[name|number]` that `[message]`
### Add a contact
`[name]`'s number is `[number]`
### Open an app
Open `[app name]`
### Open a website
Open `[website]`
<hr /><br />For those of you wondering, voítheia means "help" in Greek.
